# CadPad (Hack Club Hackpad)

This was my first hardware project and also the first time for me to use a CAD software (it ragebaited me so hard, so I named my project CadPad :D). It has 6 keys, 1 rotary encoder and 1 OLED screen. I mostly followed the guide for the beginning part but then had to read up on a lot of stuff to complete this project.
